# ngStato

State management framework-agnostic, **state-first**, with **RxJS optional**.

- Same real-world capabilities as NgRx for complex flows.
- Less boilerplate and easier DX for CRUD-heavy apps.
- `@ngstato/core` works beyond Angular.

## Quick start

```bash
npm install @ngstato/core @ngstato/angular
```

Go to [Getting Started](/guide/getting-started).
