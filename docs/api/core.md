# Core API

## Store

- `createStore()`: create a state container with actions, selectors, and effects
- `on()`: react to actions from another store

## Composition

- `withPersist()`: add persistence strategy (local/session storage)
- `withEntities()`: inject entity slice, actions, and selectors

## Entities

- `createEntityAdapter()`: normalized entity operations and selectors

## Tooling

- `connectDevTools()`: integrate with devtools for debugging
